# 亚马逊商品数据分析系统 - 需求文档

## 项目目标

通过数据驱动的方式，帮助产品选型决策，实现：
1. **发现市场空白机会**：识别竞争少、需求大的细分市场
2. **竞品深度分析**：分析头部竞品的定价、功能、评价，找到差异化切入点
3. **快速验证产品想法**：评估市场规模和竞争强度

## 核心功能模块

### 模块1：ASIN数据抓取

**输入**：关键词（默认：camping）

**数据源**：
- API抓取：`D:\Product\api\scraperAPI\src\keyword_scraper_cli.py`（search功能）
- 价格数据：`D:\Product\price\main.py`

**抓取范围**：
- 搜索结果前100个ASIN
- 每个ASIN的基础信息：标题、价格、评分、评论数、BSR排名、品牌、上架时间

**数据质量要求**：
- 去重处理（同一ASIN只保留一条）
- 异常数据标记（价格为0、缺失图片、无评分等）
- API限流处理（失败重试3次，间隔5秒）

### 模块2：卖家精灵数据补充

**参考项目**：`D:\Product\plugin\sellerspirit\main.py`

**抓取数据**：
- 关键词搜索量（月搜索量）
- 市场集中度（CR4：前4名市场份额）
- 销量估算
- 关键词扩展建议

### 模块3：AI智能分类校验

**参考项目**：`D:\Product\agent_class\product_classifier.py`

**功能**：
1. 验证亚马逊官方分类是否准确
2. 自定义场景分类（如：家庭露营、背包徒步、车载露营）
3. 识别与主题不相关的ASIN

**输出**：
- 分类异议列表（ASIN + 原因 + 建议分类）
- 不相关产品列表（偏离主题的ASIN）

### 模块4：数据整合与深度分析

**核心分析维度**：

#### 4.1 市场需求与竞争分析
- **市场规模**：关键词月搜索量、总ASIN数量
- **竞争强度**：
  - 市场集中度（CR4、CR10）
  - 头部品牌占比
  - 平均评论数（反映市场成熟度）
- **关键词扩展**：
  - 一个主关键词可延伸的子关键词数量
  - 每个子关键词对应的ASIN数量
  - 长尾关键词机会（搜索量>1000 且 竞品<50）

#### 4.2 价格与利润空间分析
- **价格分布**：
  - 价格区间分布（$0-20, $20-50, $50-100, $100+）
  - 各价格带的ASIN数量占比
  - 主流价格带识别（占比>30%）
- **定价策略**：
  - Top 10产品的平均价格
  - 价格与评分的相关性分析
- **利润空间评估**：
  - 预估成本（价格 × 30%）
  - 预估利润率（假设售价 - 成本 - FBA费用）

#### 4.3 产品生命周期分析（重点）
- **新品机会挖掘**：
  - 定义：上架时间 < 6个月 且 BSR排名 < 10000
  - 筛选条件：评论数 > 50（证明有销量）
  - 输出：Top 100 新品ASIN列表
  - 分析：新品的共同特征（价格带、功能点、设计风格）
- **趋势识别**：
  - 近6个月新品数量趋势
  - 新品平均评分 vs 老品平均评分
  - 新品主打的差异化卖点

#### 4.4 用户反馈与痛点（可选扩展）
- 评分分布（5星/4星/3星/2星/1星占比）
- 差评关键词提取（Top 10高频词）
- 未被满足的需求识别

### 模块5：可视化决策报告

**输出格式**：单文件HTML报告 + CSV详细数据

#### 5.1 报告结构

**第一部分：核心指标卡片**（一屏展示）
- 市场规模：月搜索量、总ASIN数
- 竞争强度：CR4、平均评论数
- 价格中位数、主流价格带
- 新品机会数量（近6个月）
- 市场空白指数（自定义算法：搜索量/竞品数）

**第二部分：新品机会看板**
- Top 100 新品ASIN表格（可排序、筛选）
- 新品趋势图表

**第三部分：竞品分析矩阵**
- 价格-评分散点图（气泡大小=评论数）
- 品牌集中度饼图
- 价格分布直方图

**第四部分：关键词扩展地图**
- 主关键词 → 子关键词树状图
- 每个子关键词的搜索量、ASIN数量

**第五部分：产品经理视角分析**
- 市场空白机会：哪些价格带竞争少？
- 差异化切入点：新品的共同创新点是什么？
- 风险评估：市场是否过度饱和？

**第六部分：运营视角分析**
- 推荐切入价格带
- 推荐主打关键词（搜索量/竞争度比值最高）
- 竞品监控清单（Top 10 ASIN）

#### 5.2 交互功能
- 表格支持排序、筛选
- 图表可点击查看详情
- 一键导出CSV数据

## 关键指标定义

| 指标名称 | 计算公式 | 说明 |
|---------|---------|------|
| 市场空白指数 | 月搜索量 / 竞品数量 | >100为高机会 |
| 市场集中度CR4 | Top4品牌销量 / 总销量 | >60%为高集中 |
| 新品定义 | 上架时间<6个月 且 评论数>50 | 证明有销量的新品 |
| 主流价格带 | 占比>30%的价格区间 | 市场主战场 |
| 长尾关键词 | 搜索量>1000 且 竞品<50 | 低竞争机会 |

## 使用流程

1. 修改 `config.json` 配置关键词
2. 运行主程序：`python main.py`
3. 等待数据抓取与分析
4. 打开 `report.html` 查看报告
5. 根据报告决策是否进入该市场
